# 常用接口 / 协议引脚速查卡（Markdown 版）

> 方向约定：→ 主控输出 ← 主控输入 ↔ 双向  
> 注：电平/速率以器件手册为准（下列常见值仅供参考）

---

# RGMII（千兆以太网）

| 信号      | 方向 | 功能与说明 |
|-----------|------|------------|
| TXC       | →    | 发送时钟，千兆时 125 MHz，**DDR** |
| TXD[3:0]  | →    | 发送数据 4bit，**DDR**（上/下沿各传 2bit，总 4bit） |
| TXCTL     | →    | 发送控制：**TXC 上升沿=TX_EN**；上/下沿异或可恢复 **TX_ER** |
| RXC       | ←    | 接收时钟，来自 PHY，千兆时 125 MHz，**DDR** |
| RXD[3:0]  | ←    | 接收数据 4bit，**DDR** |
| RXCTL     | ←    | 接收控制：**RXC 上升沿=RX_DV**；上/下沿异或可恢复 **RX_ER** |
| MDC       | →    | 管理时钟（MDIO 管理总线） |
| MDIO      | ↔    | 管理数据（需上拉，可多 PHY 共总线） |
