# 实数列
约定
*  $a_0$默认是0

## 一、运算

| 运算名称 | 数学表达式 | 核心特点 / 适用场景 |
|---------|-----------|-------------------|
| 加法运算 | aₙ + bₙ | 两数列对应项相加，基础组合操作 |
| 减法运算 | aₙ - bₙ | 两数列对应项相减，基础拆分操作 |
| 乘法运算 | aₙ × bₙ | 两数列对应项相乘，适配数列乘积分析 |
| 除法运算 | aₙ / bₙ | 两非零数列对应项相除，需规避零项 |
| 数乘运算 | λaₙ | 常数λ乘数列每项，数列缩放操作 |
| 差分运算（一阶） | Δaₙ = aₙ₊₁ - aₙ | 分析数列变化率 |
| 商分运算 | aₙ₊₁/aₙ | 相邻非零项作比，判定等比性（避零项） |
| 求和运算 | Σ aₙ | 前n项和，累计值计算 |
| 求积运算 | Π aₙ | 前n项积，乘积累计计算 |
| 卷积运算 | cₙ = Σₖ aₖ · bₙ₋ₖ=aₙ*bₙ| 数列叠加映射|
| 高阶差分运算 | Δ²aₙ = Δaₙ₊₁ - Δaₙ | 二阶及以上，分析凹凸性、高阶递推 |
| 移位运算 | aₙ₊ₖ 和 aₙ₋ₖ | 左移和右移，递推方程变形、信号延迟分析 |

| 离散梅林变换 | 离散变换 | 信号处理 |
## 三、公式定理

### 幂的差分


$$\Delta n^k = \sum_{i=1}^{k} (-1)^{i+1} \binom{k}{i} n^{k-i}$$


### 高阶差分


$$\Delta^k a_n = \sum_{i=0}^{k} (-1)^{i} \binom{k}{i} a_{n-i}$$


### 差分求和
$$\sum \Delta a_n = a_{n+1} - a_1$$
### 求和差分
$$\Delta \sum a_i = a_{n+1}$$

### 母函数中卷积公式
这是[对偶原理](https://blog.csdn.net/qq_26074053/article/details/149341925)的一个例子,其中母函数G是对偶映射, 需要补充定义 $G(*)=×$
对于公式$$(1-x)×(1+x^1+x^2+....+...)=1$$
中的$(1-x)和 (1+x^1+x^2+....+...)和1$是三个不同的形式幂级数,
母函数是实数列和形式幂级数之间的一个双射,也是一个对偶映射。
$$G(a_n*b_n) =G(a_n)×G(b_n)=A(x)B(x)$$
